
# Rehab Outcomes Tracker

This is a Streamlit app for tracking rehabilitation outcomes across multiple conditions.

## Features
- Patient Management
- Condition-specific Assessment Templates
- Performance-Based Test Tracking
- Progress Visualization with Charts
- Report Generation

## How to Deploy on Streamlit Cloud
1. Create a GitHub repository and upload these files.
2. Go to https://streamlit.io/cloud and sign in.
3. Click "New app" and connect your GitHub repo.
4. Set the main file path to `rehab_outcomes_tracker.py`.
5. Click "Deploy".

See `requirements.txt` for dependencies.
